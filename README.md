ForgeBox 3D
The Ultimate Studio Mesh Organizer & Previewer
ForgeBox 3D is a sleek, lightning-fast web application built specifically for 3D printing enthusiasts, digital fabricators, and creators to organize, catalog, and preview their STL asset collections entirely in the browser.

Say goodbye to cluttered sidebars and confusing file names. ForgeBox 3D delivers a high-performance desktop-class experience that lets you drag, drop, catalog, and interact with your 3D models smoothly on a digital build plate.

🚀 Key Features
Matte Visual Workspace: An immersive 3D canvas powered by Three.js featuring responsive lighting, cinematic theme transitions (Light/Dark mode), and real-time wireframe toggles.

The Fullscreen Library Hub: A beautiful, full-width dashboard overlay that separates your models into custom visual collections (e.g., Fidgets, Miniatures, Tools) so you can manage your library without cluttering your workspace.

Dynamic Asset Badges: Instantly see exactly how many models are stored in each collection with automatic item counters.

Drag & Drop Staging: Toss any .stl file directly into the interface to instantly render, inspect, and scale the model before committing it to your storage.

Local-First Offline Storage: Built on IndexedDB technology, your files and folder layouts are saved directly onto your device securely—meaning fast load times with zero server delays and absolute privacy.

🛠️ Built For Creators
Whether you are keeping track of articulated models, fidget designs, custom structural parts, or point-of-sale displays, ForgeBox 3D keeps your workspace clean so you can focus on creating.

No accounts, no cloud fees, no clutter. Just drag, sort, and preview.
